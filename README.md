## Breakout - Panupong Saisopon

โปรเจกต์ **Breakout** จัดทำขึ้นเพื่อเป็นผลงานส่งในรายวิชา  
**040223102 การเขียนโปรแกรมคอมพิวเตอร์สำหรับคณิตศาสตร์ 2 (2566)**

ต้นแบบมาจากเกม Breakout ของเครื่องเกม Atari โดยเกมมีระบบดังนี้:  

- ผู้เล่นจะเป็นไม้ที่อยู่ข้างล่าง สามารถเดินได้แค่ซ้ายกับขวาเท่านั้น  
- การเล่น 1 ครั้ง ผู้เล่นจะมีลูกบอลทั้งหมด 3 ลูก (ชีวิต) โดยจะถูกปล่อยออกมาทีละลูก  
- ผู้เล่นต้องรับลูกบอลเพื่อให้ลูกบอลไปชนกับบล็อกที่อยู่ข้างบน  
- การจบรอบจะจบเมื่อผู้เล่นชนบล็อกจนหมดด่าน  

โปรเจกต์นี้สร้างด้วย **Windows Form C# .NET** เพื่อเสริมสร้างความเข้าใจภาษา C#
